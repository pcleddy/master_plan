# Milestones

Use this directory for major findings only.

Good milestone entries:

- a Mathlib dependency or API discovery that changes the project structure
- a completed foundational theorem block
- a proof that a planned path is blocked or materially harder than expected
- a research finding that changes the roadmap

Suggested naming:

- `YYYY-MM-DD-bootstrap-complete.md`
- `YYYY-MM-DD-bounded-observables.md`
- `YYYY-MM-DD-spectral-theorem-gap.md`
- `YYYY-MM-DD-a6-obstruction.md`
